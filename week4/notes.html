Chapter 15

Frameworks

A framework is a library of JavaScript code that provides several methods that make it easier to achieve common tasks.

jQuery is the most popular of all the JavaScript frameworks.

When to Use a Library

It seems that it can provide only the parts of methods you need on a module-by module basis. But it seems it can also be a negative to use them.

MVC Frameworks

Model-View-Controller (MVC) is a design pattern that's been used for a long time in server-side languages.

MV* Frameworks

One of the main features of MVC frameworks is data binding, which is the process of linking the model and view together. As a result, a large amount of boilerplate controller code is not needed as the framework takes care of it all in the background for you.
